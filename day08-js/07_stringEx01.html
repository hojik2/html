string : 가장 많이 다루는 데이터 타입

  • java에서 사용하는 String method와 거의 유사함.
  • 문자열 데이터는 index(0부터 시작)를 가지고 있음.
  • 문자열 값 내에서 (br) 개행 하는것을 추천하지 않음
  • new String() : 객체 타입으로 선언가능 / 추천하지 않음.
  • length : 문자열의 총길이
  • 추출 관련 메서드 : substring(), substr(), slice()
  • 문자대체 관련 메서드 : replace(old, new) => 처음 찾은 값만 사용가능
  • trim() : 양쪽 공백 제거
  • padStart(길이, 채울문자) / padEnd(길이, 채울문자) 길이 만큼 문자를 채워주는 함수 010101-4****** / ******시
  • charAt(index) : 한글자 추출 / indexOf(문자) : 해당 문자의 위치 리턴
  • startWith(문자) / endWith(문자) 문자열 앞, 뒤에 문자가 존재하는지 확인하는 메서드 return true / false
