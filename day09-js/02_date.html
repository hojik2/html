날짜 관련 함수

  • new Date(); => 오늘 날짜 반환 new Date(string); => string 날짜 형식으로 변환
  • getFullYear(); 연도만 리턴
  • getMonth()+1; => 0월부터 11월까지 +1
  • getDate(); => 1~31 리턴
  • getDay(); => 요일을 리턴 0(일)~6(토)
  • getHours(), getMinutes(), getSeconds() : 시분초 추출
  • toLocalDateString() : 2024. 08.07 toLocalTimeString() toLocalString()
